# Far Cry Modding

This organization aims to become a unified technical knowledge base for modding the Far Cry series (FC1–FC6) — documentation, reverse-engineered file format specs, and preserved tool mirrors.

## Want to help?

If you're an active member of the Far Cry modding community and would like to help build a centralized modding knowledge base, please [open an issue](https://github.com/FarCry-Modding/.github/issues) introducing yourself and what you'd like to contribute.
